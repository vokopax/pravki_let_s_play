игра "Виселица" с расширенным функционалом
Обоснование новых функций в игре "Виселица"
Добавленные функции:
1. Угадывание слова целиком
   def get_player_input_with_word_guess(used_letters, secret_word):
   Данная функция делает более естественным игровой процесс, т.к. в классической "Виселице" игроки часто пытаются угадать слово, если догадались
   Дает стратегический выбор (игрок решает рискнуть угадать слово или продолжить по буквам)
   В целом ускорение игры (если игрок догадался, он может сразу победить)
Функция расширяет существующую get_player_input(), не ломая её, сохраняет все существующие проверки ввода


2. Система подсказок (1 раз за игру)
   def give_hint(secret_word, guessed_letters):
   Данная функция предоставляет помощь в критический момент (подсказка предлагается при 2 или меньше попытках)
   Ресурс ограниченный и 1 подсказка на игру создает стратегический выбор
   Предоставляет контекстную помощь (показывает конкретную букву и её позицию)
   Опциональна (игрок может отказаться от подсказки)
Функция интегрирована в основной игровой цикл без нарушения логики и использует существующие функции check_letter_in_word() и update_display_word()


3. Счётчик времени игры
   start_time = time.time()

   end_time = time.time()
   game_duration = end_time - start_time
   Данная функция добавляет измерение прогресса (игрок видит, сколько времени потратил)
   Соревновательный элемент (можно пытаться побить свой рекорд)
   И также обратную связь (разные сообщения в зависимости от времени)
Функция не влияет на игровой процесс, добавляет информацию к существующей статистике
